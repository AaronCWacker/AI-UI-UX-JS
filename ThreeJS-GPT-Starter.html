Skyforge

WebGPU Academy

This version fixes the broken WebGPU import pattern, uses the WebGPU build with TSL nodes, and turns the game into an educational 3D playground. Each shrine teaches a real Three.js concept while expanding the world.

Modern Import MapMaps three and three/webgpu to the WebGPU bundle, plus three/tsl for node shading.
Node MaterialsAnimated glow and wobble with MeshStandardNodeMaterial, color(), normalLocal, sin(), time, and positionLocal.
Interactive LessonsRaycasting lets the player click shrines to trigger scene-graph demos and build geometry visibly.
Game FeelInstanced swarms, curve couriers, floating shards, procedural terrain, and a central forge nucleus.
