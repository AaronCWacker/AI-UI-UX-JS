How to Find Your App's API Details

  1. Open your Streamlit/Gradio app in a new browser tab.
  2. Open your browser's Developer Tools (usually F12 or Ctrl+Shift+I) and select the "Network" tab.
  3. Perform an action in your app's UI (e.g., type in a textbox and click a submit button).
  4. Look for a new request in the Network log. It's often named `predict`, `run`, or `call`. Click on it.
  5. Endpoint URL: In the "Headers" tab of the request details, find and copy the full Request URL. Paste it into the "Endpoint URL" field below.
  6. Payload: Switch to the "Payload" or "Request" tab to see the JSON data. This will show you the correct `fn_index` and the structure for the `data` array you need to send.

Configuration
