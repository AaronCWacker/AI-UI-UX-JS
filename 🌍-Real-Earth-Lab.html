🌍
Pivoted Focus
This version is about visual quality first: realistic earth shading, better ocean behavior, and a clean transparent layer stack.
🌊
Sea Layer
The ocean sheen uses a transparent sphere driven by the specular map so water catches light while land stays quieter.
☁️
Cloud Layer
Clouds are their own shell so you can fade or hide them independently for a clear map look or a cinematic weather look.
🗺️
Map Layer
Turn on the transparent graticule overlay when you want a more atlas-style map read instead of pure realism.
🛰️
Runtime
Static HTML + JS only. No build step, no backend, no Docker. WebGPU renderer with textures loaded directly in the browser.
📚
Texture Sources
Uses public earth texture URLs from the Three.js examples site for diffuse, normal, and specular earth maps. If you later want even more realism, we can swap in larger NASA or Blue Marble layers and add night lights.
🧭 Pivot complete: this build prioritizes a better earth render over the earlier semantic demo. Use the transparent layer toggles to move between cinematic globe and atlas-style map presentation.
